week-2 variables,constants and data types

Variables are containers used to store data values. Think of them as labeled boxes.

three ways to declare variables are:

  1. var: is a way of declaring variables that are reusable
  2. let: same as var is a way of declaring variables that are reusable
  3. const: declares variables that can only be assigned once eg const = l
